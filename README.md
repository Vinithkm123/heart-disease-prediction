# Heart Disease Prediction using Logistic Regression

This project predicts the presence of heart disease using a machine learning model.

## Dataset
- Source: Heart Disease Dataset
- Rows: 1025
- Features: 13
- Target: target (0 = No disease, 1 = Disease)

## Methodology
- Data loading and exploration
- Train-test split (80-20)
- Logistic Regression model
- Model evaluation

## Results
- Accuracy: 79.51%
- Precision: 75.63%
- Recall: 87.38%

## Tools & Libraries
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
